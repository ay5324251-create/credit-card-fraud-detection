# 💳 AI Credit Card Fraud Detection System

An AI-based Credit Card Fraud Detection System built using Python, Scikit-learn, and Streamlit.

## 🚀 Features
- Detects whether a credit card transaction is Fraud or Normal
- Machine Learning model using Random Forest Classifier
- Interactive user input with Streamlit sidebar
- Displays fraud probability
- Dataset preview and fraud distribution chart

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## 📂 Project Structure
credit-card-fraud-detection/
│── app.py
│── creditcard.csv
│── README.md

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install pandas streamlit scikit-learn
3. Run the app:
   streamlit run app.py

## 📊 Output
The app allows users to enter transaction details and predicts whether the transaction is fraudulent or normal.
